## 🐰 구현할 기능 목록

### 입력 기능

1. [] 경주 할 n대의 자동차 이름(이름은 쉼표(,)로 구분)
    3. [] (예외) 이름이 5자 초과일 경우
    4. [] (예외) 쉼표 이외에 구분자 입력
2. [] 자동차 경주 게임을 시도할 횟수
    3. [] (예외) 숫자가 아닌 문자 입력
    4. [] (예외) 아예 입력하지 않은 경우

### 출력 기능

1. [] 이름을 쉼표로 구분하여 전진하는 자동차를 출력
2. [] 전진할 때, " - " 출력
3. [] 우승자 출력
    4. [] 우승자가 1명 이상일 경우에 쉼표(,)를 이용

### 과정

1. [] 시도할 횟수만큼 자동차 경주 실행
2. [] 1회당 각 자동차마다 0-9 사이의 무작위 값 구하기
    3. [] 무작위 값이 4 이상일 경우 전진
4. [] 게임 완료 후 우승자 구하기
    5. [] 각 자동차의 " - " 개수 구하기
